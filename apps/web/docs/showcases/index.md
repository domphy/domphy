# Showcases

Real-world apps built with Domphy. Every interface below runs on `@domphy/core`, `@domphy/theme`, and `@domphy/ui` — no React, no Vue, no virtual DOM.

---

## Domphy Docs — domphy.com

This documentation site is itself the primary showcase — a full SSG docs engine built entirely with Domphy, with no third-party UI framework.

| | |
|---|---|
| **Markdown → Domphy** | `@domphy/markdown` |
| **Docs engine (SSG + routing)** | `@domphy/press` |
| **SSR + streaming** | `@domphy/app` |
| **Code editor islands** | `@domphy/core` + CodeMirror |
| **Live demos** | `@domphy/core` client-side hydration |
| **Diagrams** | `@domphy/mermaid` |
| **Search** | `@domphy/core` + custom inverted index |
| **UI** | `@domphy/ui`, `@domphy/theme` |
| **Link** | [domphy.com](https://domphy.com) |
